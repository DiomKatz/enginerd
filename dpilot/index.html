DPilot

Advanced Flight Telemetry & Control System
Project

A personal research project exploring flight data logging and control systems. Features include state communication, data logging, web-based monitoring, and video processing for educational and experimental purposes.

Project Features

🔄 Data Architecture

  • Basic telemetry transmission
  • State packet communication
  • Data synchronization testing
  • Simple command interface

📡 Communication

  • LoRa radio transmission
  • Basic encryption implementation
  • Local range testing
  • Experimental packet structure

🌐 Web Interface

  • Always-on WiFi hotspot mode
  • Live telemetry dashboard
  • Smartphone-accessible control
  • Remote recording management
  • Real-time attitude display

🎬 Video Recording

  • Raw and overlay recording modes
  • Perfect frame synchronization
  • Artificial horizon overlay
  • G-G acceleration diagrams
  • CSV data correlation
